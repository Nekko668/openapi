row_countYInt每页的行数

返回参数

以下返回参数列表仅列出了返回参数中Data里的JSON 数据参数，公共返回参数请参见签名验证章节的 公共参数说明。

参数名类型说明
page_countInt分页总的页数
total_countInt总数
row_count每页的行数
page_currentInt当前第几页
arr_deviceArray设备数组

设备数组

参数名类型说明
+ catalog_idInt目录ID
+ device_idInt设备ID
+ device_nameString设备帐号
+ device_snString设备标识
+ device_typeInt设备类型
+ device_subtypeInt子类型
+ device_LongitudeDouble经度
+ device_latitudeDouble纬度
+ device_aslDouble海拔
+ device_addressString地点名称
+ device_rlocationXDouble室内定位X坐标
+ device_rlocationYDouble室内定位Y坐标
+ device_rlocationZDouble室内定位Z坐标
+ has_panInt是否支持云台 0否 1是
+ has_videoInt是否有视频能力 0否 1是
+ has_audioInt是否有语音对讲 0否 1是
+ user_idInt绑定的帐号ID
+ onlineInt是否在线 0否 1是
+ room_idArray/Int在会列表 0 不在会 []在会列表

请求示例

json
{
  "catalog _id": 5,
  "row_count": 0,
  "page_index": 0
}
// 示例参数
param = {
  "catalog _id": 5,
  "row_count": 0,
  "page_index": 0
}

// 将param进行Base64编码

// 用appkey对MD5的数据进行HmacSHA1
